Projetos em HTML, CSS e JS

1) Escreva um programa em HTML, CSS e JS que mostre na tela a mensagem "Olá, Mundo!"

2) Faça um programa em HTML, CSS e JS que leia o nome de uma pessoa e mostre uma mensagem de boasvindas para ela:

Ex:

Qual é o seu nome?

João da Silva Olá João da Silva, é um prazer te conhecer!

3) Crie em HTML, CSS e JS um programa que leia o nome e o salário de um funcionário, mostrando no final uma mensagem.

Ex:

Nome do Funcionário: Maria do Carmo

Salário: 1850,45

O funcionário Maria do Carmo tem um salário de R$1850,45 em Junho.

4) Desenvolva em HTML, CSS e JS um algoritmo que leia dois números inteiros e mostre o somatório entre eles.

Ex:

Digite um valor: 8

Digite outro valor: 5

A soma entre 8 e 5 é igual a 13.
